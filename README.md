## Setup & Installation

1.  Clone the repository:

    ``` bash
    git clone https://github.com/yourusername/portfolio.git
    cd portfolio
    ```

2.  Install dependencies:

    ``` bash
    npm install
    ```

3.  Run the development server:

    ``` bash
    ng serve
    ```

Visit `http://localhost:4200` in your browser to see the portfolio in
action.

## Deployment

Build for production:

``` bash
ng build --prod
```

Deploy the `/docs` folder to your preferred hosting platform (GitHub
Pages, Netlify, Vercel, etc.).
